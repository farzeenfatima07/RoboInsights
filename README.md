# RoboInsights

About RoboInsights
RoboInsights is a NotebookLM-style research notebook built for robotics competition coaches and educators. Instead of digging through scattered rulebooks, scoresheets, and coaching notes every season, coaches can upload their competition documents and get instant, source-grounded answers, summaries, and analytics — all in one place.
What it does

Upload sources — rulebooks, scoresheets, coaching notes, and results data (PDF/CSV)
Ask questions — get answers grounded directly in your uploaded documents, with inline citations back to the source
Generate briefs — one-click summaries like rule-change reports, scoring trend analyses, and team prep checklists
Visualize performance — auto-generated charts showing score distributions, round-by-round breakdowns, and team progress over time

Why I built this
As a robotics coach and STEAM educator with a background in data analytics, I wanted a tool that combines both — something that treats competition data (scores, rule changes, judging criteria) with the same rigor as any other dataset, while staying genuinely useful for the day-to-day work of coaching. RoboInsights is built on real competition formats and sample data modeled after WRO and RoboFest events.

Tech stack
-Vanilla HTML, CSS, and JavaScript (no framework dependencies)
-Chart.js for data visualization
-Designed to plug into an LLM backend (e.g., Claude or Gemini API) for source-grounded Q&A

Status
This is currently a frontend prototype with mock data, built as a portfolio project demonstrating the intersection of robotics competition expertise and data analytics tooling.
